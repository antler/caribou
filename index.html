caribou by antler

Caribou is a dynamic web application generator with automatic model and route creation.

Caribou was designed to remove the extraneous effort usually required to develop a web application. By focusing all effort into developing features that can be reused over and over, you need only specify what your application is supposed to do.

Dependencies

Caribou depends on: clojure - http://github.com/richhickey/clojure leiningen - http://github.com/technomancy/leiningen

Install

Install Java >= 1.6 Install leiningen: http://github.com/technomancy/leiningen Create a new Caribou project: lein new projectname
