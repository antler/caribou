caribou

Caribou is a dynamic web application generator with antlers

Caribou was designed to remove the extraneous effort usually required to develop a web application. By focusing all effort into developing features that can be reused over and over, you need only specify what your application is supposed to do.

Dependencies

Caribou depends on:
clojure
leiningen

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/antler/caribou

Install

Install Java >= 1.6
Install leiningen
Create a new Caribou project:

$ lein caribou new $sitename
